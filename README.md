# Mumbai Food Delivery Market Analysis

## Overview
Interactive Power BI dashboard analyzing 1,000+ restaurants 
across Zomato & Swiggy in Mumbai.

## Tools Used
- Power BI
- Excel / CSV

## Key Insights
- Italian and Sushi/Japanese generate highest revenue in Mumbai
- Vashi and Kurla have highest monthly order volumes
- Higher rated restaurants show upward trend in monthly orders
